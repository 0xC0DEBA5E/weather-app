Electron

  • Open Source Library von GitHub
  • Desktop-Applikation erstellen mit HTML, CSS, JavaScript
  • Applikationen laufen unter Mac, Windows und Linux

app Module

  • Läuft im Main-Prozess
  • Verwaltet den Lifecycle der Applikation
  • Events können mehrere Parameter entgegenehmen
  • Jedes Events gibt unterschiedliche Parameter zurück
                
const {app} = require('electron')
app.on('event', (... parameters ...) => {
    // do something
})
