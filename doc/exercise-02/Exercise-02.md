## Locales

With `app.getLocale()` you can get the users locale-setting from within the main process. 
Possible values are i.e. `de`, `de-DE`, `en`, `en-US`, ...
For more possible values, have a look at: [official documentation for locales](https://github.com/electron/electron/blob/master/docs/api/locales.md).
